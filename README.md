# BookStore
Bookstore is a simple RESTful API Java Spring Boot application to store and sell books.

---
# Tools/Framework
<ul>
<li>Java 17</li>
<li>Spring Boot 3.30</li>
<li>PostgreSql</li>
<li>Java Mail Sender</li>
<li>Paypal (payment gateway)</li>
<li>SpringDoc</li>
</ul>
